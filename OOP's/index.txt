#default Constructor
 Agar Default Constructor ko hum nahi banayenge to unhe Python apne aap bana dega 
 A default constructor is one that takes no arguments other than the mandatory self reference. 

 class Dog:
    def __init__(self):
        self.species = "Canine" # Initializes all objects with the same value
        self.sound = "Woof"


2. Parameterized Constructor
A parameterized constructor accepts one or more arguments in addition to self. This type allows you to pass specific
 data at the time of object creation, ensuring each object can have unique initial values. 

 class Dog:
    def __init__(self, name, age):
        self.name = name # Initializes objects with unique, custom values
        self.age = age

# Creating instances with specific values
dog1 = Dog("Buddy", 3)
dog2 = Dog("Max", 5)
